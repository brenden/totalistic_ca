Totalistic cellular automata are a class of cellular automata whose next state is determined solely by the current state and the number of live cells in the neighborhood. Two values define every totalistic ruleset: the number of live neighbors required for a dead cell to become live, and the number required for a live cell to remain so. The B/S notation is commonly used to describe such rulesets. B3/S23 means that three live neighbors allow a dead cell to become live, and two or three live neighbors allow a live cell to persist. B3/S23 actually describes the famous Conway's Game of Life ruleset

The shape of the surface on which the cellular automata live affects the system's outcome. Rectangles and tori are common choices, but more exotic, non-orientable surfaces such as Klein bottles can be used as well. By specifying the horizontal and vertical folds, many different surfaces can be created. For example, vertical:1, horizontal:0 yields an open cylinder; horizontal:1, vertical:-1 yields a Klein bottle.
